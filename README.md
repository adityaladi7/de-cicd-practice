🚀 Automated Data Pipeline with CI/CD and Data Validation

## Overview
A Python-based data pipeline with CI using GitHub Actions.
It validates data, runs transformations (ETL), and tests outputs automatically on every push.

## Pipeline Flow
Lint → Validate Data → Run ETL → Test Output → Build

## Tech Stack
- Python (pandas, pytest, flake8)
- GitHub Actions (CI/CD)

  ## Key Features
- Automated data validation (null checks, uniqueness)
- ETL pipeline with transformation logic
- CI/CD pipeline using GitHub Actions
- Automated testing of transformed data

## Why this project

This project demonstrates a real-world data engineering workflow where data quality checks, transformations, and validations are automated using CI/CD pipelines.
